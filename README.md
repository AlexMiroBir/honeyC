1. For start the server you need to have relevant config/env.json file with your settings (not in git)

Example:
{
  "server": {
    "port": 443,
    "host": "localhost"
  },
  "db": {
    "port": 27017,
    "host": "127.0.0.1"
  }
}

2. Command to download node modules: <b>npm install</b>
3. Command to start server: <b>npm run dev</b>
4. Api-docs url:<b> https://localhost/api-docs</b>
